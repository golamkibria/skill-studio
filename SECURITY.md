# Security Policy

## Supported Versions

This project is currently maintained on the `main` branch. Security fixes are applied to the latest code only.

## Reporting a Vulnerability

Please report security issues responsibly:

1. Do not open a public GitHub issue for a suspected vulnerability.
2. Use GitHub Security Advisories (preferred):  
   Repository -> `Security` -> `Report a vulnerability`
3. If advisories are unavailable, contact the maintainer directly:
   - GitHub: https://github.com/golamkibria

Include the following in your report:

- A clear description of the issue
- Steps to reproduce
- Potential impact
- Suggested fix (if known)

## Disclosure Process

- You will receive an acknowledgment as soon as possible.
- The issue will be validated and prioritized.
- A fix will be prepared and released.
- Public disclosure should happen after a fix is available.
